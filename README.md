## How to connect the motors

The prefixed M stands for motor (probably).
Ports 1-4 should be connected to the **S**1-4 ports on the BrickPi.

* MA = Pen motor
* MB = X-axis motor
* MC = Y-axis motor
* MD = Y-axis motor
* PORT_1 = Unused
* PORT_2 = Unused
* PORT_3 = Unused
* PORT_4 = Unused
